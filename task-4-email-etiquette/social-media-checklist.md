


## ✅ Part B — Social Media Do's and Don'ts

| | Rule |
|---|------|
| ✅ | Keep LinkedIn and GitHub professional — recruiters actively check these |
| ✅ | Think before you post — would your professor or future employer be comfortable seeing it? |
| ✅ | Use privacy settings on personal platforms to control who sees your content |
| ✅ | Credit others when you share their work, ideas, or images |
| ✅ | Report cyberbullying using the platform's built-in tools immediately |
| ❌ | Don't post complaints about college, faculty, or classmates publicly |
| ❌ | Don't share your college ID or registration number on any public platform |
| ❌ | Don't accept requests from unknown people on professional platforms |
| ❌ | Don't forward unverified news or rumours without checking the source |
| ❌ | Don't use social media to settle personal disputes — always go private |
