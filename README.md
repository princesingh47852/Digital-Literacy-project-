## 📖 About This Repository

This repository is my **Digital Literacy Portfolio**, submitted as part of CSE0001 at VIT Bhopal University. I took on the role of a **Student Digital Ambassador** — building and documenting the digital skills every first-year student needs: creating a professional online presence, communicating formally, practising coding, and staying safe online.

The portfolio is organized across **five tasks**, one per course module. All files are here — task write-ups, screenshots, email drafts, checklists, and a full project report.
